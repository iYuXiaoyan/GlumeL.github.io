前言

免费 免费 免费

简说博客

一般新手搭建博客都是用服务器搭建，或者使用平台的博客
平台例如：CSDN 新浪 博客园 简书 … 甚至知乎。
PHP程序：Typecho WordPress ….很多
静态程序：Hexo Jekyll Simple …很多

个人主推Hexo

个人原因

Hexo特点：免费，基本无成本，无服务器运行，多端。

经常维护服务器麻烦，静态托管大厂，无需维护。

服务器快到期，没啥子钱续费（没钱）

操作简单，新人易懂，多端运行，无需维护。全静态

图片托管github和jsdelivr基本全部白嫖。总之就是香！！！

静态文件引用外链，速度杠杠的！

Jekyll也是免费为什么不推荐

  1. Jeky基于Ruby实现，安装Jeky需要搭建Ruby环境，在Windows搭建Ruby环境并不是被推荐的，而 Hexo基于NodeJs实现，在Windows上安装NodeJs开发环境简单。
  2. 实现简单，只需几步(本地预览舒服)
  3. 两者都支持Markdown语法，这点我非常喜欢。
  4. Hexo 主题多样化，自定义程度高

简说

教程分为三个部分

  1. Hexo搭建和部署到Coding和Github Pages双端，实现国内分流解析加快浏览速度，以及绑定个人的域名
  2. Hexo本地部署和主题更换配置，外加我的个人美化(开源)
  3. Hexo的各类插件安装与基本命令，访问统计，百度收录检测
  4. 使用多平台CDN进行加速流量，和Github+Jsdeliver进行搭建图床

Hexo简介

Hexo是一款基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Coding上，是搭建博客的首选框架。大家可以进入hexo官网进行详细查看，因为Hexo的创建者是台湾人，对中文的支持很友好，可以选择中文进行查看。

Hexo搭建

使用本地配置Hexo，和使用Coding和Github Pages进行搭建，并推送发布，分流解析和绑定个人域名

搭建前如需绑定个人域名，需有一个域名，建议去DNSPOD进行购买，最低仅1元/年
所需工具：电脑，脑子，手，良好的网络，和一个能翻过海的梯子

搭建步骤

  1. 安装Git

  5. 生成SSH添加到GitHub，Coding

这个时候它会告诉你已经生成了.ssh的文件夹。在你的电脑中找到这个文件夹。

ssh，简单来讲，就是一个秘钥，其中，id_rsa是你这台电脑的私人秘钥，不能给别人看的，id_rsa.pub是公共秘钥，可以随便给别人看。把这个公钥放在GitHub上，这样当你链接GitHub自己的账户时，它就会根据公钥匹配你的私钥，当能够相互匹配时，才能够顺利的通过git上传你的文件到GitHub上。

而后在GitHub的setting中，找到SSH keys的设置选项，点击New SSH key
把你的id_rsa.pub里面的信息复制进去。
