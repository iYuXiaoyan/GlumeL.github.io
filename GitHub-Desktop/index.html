GitHub客户端

前段时间我想把自己的一些东西上传到github上，学习了一段时间git，但是觉得有点麻烦，然后就去查了有没有客户端，没想到还真有(其实出来好久了)，最后找到了比较好用的工具GitHub Desktop。在此做出详细记录

Github Desktop官网地址：https://desktop.github.com/

上面官网下载很慢，没扶墙的兄弟们可以下载这个–Glume Cloud

既然讲，那就重头讲 ! !

第一步 注册账户

Github官网：https://github.com 下图

  1. 我们点击Sign up 注册一个github账户

  2. 首先是用户名，电子邮箱，密码，然后验证一下即可注册.

  1. 然后选填你的兴趣和作用

  1. 提示你需要验证你的邮箱，验证一下就行了

  2. 到这里基本注册完成了

第二步 创建仓库

  1. 点击顶部的加号，选择 New repository 创建一个新的仓库，下图

  2. 进入到新建仓库页面，下图我们先新建一下仓库名字(按用途)然后设置为Public(公开)，并建立一个README(描述文件)(方便下文操作)

    • 首先后续可改为私人，自行摸索
    • 然后仓库名也是可更改

  3. 基本到这一步，我们网页端操作基本完成了 ，下图

第三步 下载客户端

前面也说了如何下载

强烈推荐: 下载很慢，没扶墙的兄弟们可以下载这个–Glume Cloud

官网速度(不翻墙进不去)

我的速度(还好手速够快)

下载完成后为.exe格式可执行文件，我们进行安装

  1. 安装完成后我们进行登录，点击File

  1. 再点击Flie菜单中的Options

  1. 选择前往Github.com进行登录，如果登录不了那就，手动输入刚刚注册的账号密码

  2. 登录成功后如下图，显示出了我们的测试仓库

  3. 选中我们的测试仓库，进行克隆到本地

  4. 不建议克隆到系统盘，点击Clone进行克隆

  1. 如果东西多的话，肯能会很慢，建议扶墙

  2. 克隆到本地完成会返回到这个节面

  3. 详情讲一下这个界面的主要功能

拉取新的github仓库，点击File目录下的Clone a repository

URL获取方法 进入网页端https://Github.com/GlumeL 进入刚刚新建仓库点击Code，点击复制小图标

填入URL地址自动填充目录，点击Clone，进行拉取克隆

记得扶墙(等会推荐个加速方法)

  1. 选中仓库，选择本地资源管理器打开

  2. 打开后对文件进行修改，添加，删除操作

修改文件进行保存，进入github的客户端

  1. 进行推送，刚刚修改的文件，推送到github仓库，点击Push origin

  2. 推送完成，如果网页端进行更改如何重新拉取仓库文件办法

13.点击View on Github 在网页端查看，点击后会跳转到网页端，推送完成

基本操作基本结束了，然后说了怎么多，我后续发现了汉化教程，你说狗不狗

第四步 汉化客户端

首先我们前往– 汉化包

下载汉化所需要的文件

声明以下几点

  • 本人不保证所有版本的github客户端都能通用
  • 第二，操作失误，造成丢失数据文件，不要找我
  • 第三，我本着分享精神，分享，勿喷我
  • 第四，操作问题发生错误不要找我
  • 第五，以下仅为演示，版本要注意对应，汉化有工具，可一键

操作开始

  1. 找到桌面上的Github Desktop 文件，右击点击打开文件所在位置

  2. 目录形式应该是这样的，这里2.5.3和2.5.4为版本，测试都可以使用打开目录app-2.5.4//app-2.5.3，再进入resources，再进入app文件夹，将文件拖入进入，如果提示替换就替换掉

  3. 具体目录为GitHubDesktop\app-2.5.3\resources\app

  4. 重新打开Github客户端，我们发现白屏了(不要慌[其实慌]，说明不支持2.5.4)，我的操作问题，重新安装Github Desktop，那就重新操作，我们替换好文件，重新打开

  5. 汉化完成，符合了国人操作 ，，这玩意原来是摘要啊

网页端操作介绍

这是个坑，先讲一下如何删除你的账号吧

  1. 网页端点击的你的头像，点击Settings 设置
  2. 点击左侧第二行Account，下拉到最后，你会看见一个红色字体的按钮Delete your account
  3. 点击它，输入你的账号信息
  4. 删除完成

正常点

因为刚刚我们测试创建了一个仓库，而我又用不到，那我就要删除他

  1. 先进入我们的仓库，点击Settings

  2. 下拉到最后，红色字体的按钮，点击最后一个Delete this repository

  3. 按照他的要求输入用户名/仓库来确定

  4. 删除完成 Your repository “GlumeL/test” was successfully deleted.
